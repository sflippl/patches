Color contrasts¶

Now we want to predict the different color channels from each other. This means defining a data type with corresponding datasets that are not a time series. Whereas clearly a multi-view perspective on such a problem appears richer, we are content here with a directed estimation.

n_epochs = 10
with tqdm(total = n_epochs*len(cdl)*len(lst_cpc)) as pbar:
    for cpc in lst_cpc:
        loss_traj = []
        optimizer = optim.Adam(cpc.parameters(), lr=0.01)
        for epoch in range(n_epochs):
            running_loss = 0
            for i, data in enumerate(cdl):
                if i<len(cdl):
                    optimizer.zero_grad()
                    code = cpc(data)
                    loss = criterion(code)
                    loss.backward()
                    optimizer.step()
                    running_loss += loss
                    if i % 5000 == 4999:
                        loss_traj.append(running_loss.detach().numpy()/500)
                        running_loss = 0
                    pbar.update(1)
        lst_loss_traj.append(loss_traj)
