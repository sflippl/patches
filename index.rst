patches: Contrastive coding in Pytorch
================================================================================
.. image:: https://travis-ci.com/sflippl/contrastive-predictable-coding.svg?token=29zpy5Q2m7UyeivkA9x5&branch=master
    :target: https://travis-ci.com/sflippl/contrastive-predictable-coding
.. image:: https://img.shields.io/github/license/mashape/apistatus.svg?maxAge=2592000
    :target: https://github.com/sflippl/predicode/blob/master/LICENSE)
.. image:: https://img.shields.io/badge/python-3.6%20%7C%203.7%20%7C%203.7--dev-blue

'patches' implements contrastive coding using torch, allows an easy comparison
with alternative supervised and unsupervised learning algorithms, and provides
example datasets.

You can install patches using

>>> pip install git+https://github.com/sflippl/patches

or alternatively get started online by clicking on the Colab badge on the
tutorial pages.
